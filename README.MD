# next-test

## Getting Started

superplate is a Next.js all-in-one project generator. Create your project with the tools you need without spending hours on setting them up.

## Available Scripts

### Initial setup.

```bash
    cp .env.example .env
```

### Running the development server.

```bash
    yarn dev
```

### Running the production server.

```bash
    yarn start
```

## Commit Rule

### Convention check

```bash
    yarn prettier
    yarn lint
```
